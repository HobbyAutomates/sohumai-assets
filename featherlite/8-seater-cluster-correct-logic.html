8-Seater Cluster — Horizontal orientation (screens run LEFT → RIGHT)
4 screen panels, each 4 ft wide, running left → right along the centre join line.
Each screen sits on top of the desk surface at the join, giving privacy between the person above and the person facing them below. One screen per facing pair = 4 screens total for 8 people.
Same cluster rotated 90° — screens now run TOP → BOTTOM
What changed when rotated 90°:
📐 Cluster dimensions flip:
Was 16 ft wide × 4 ft tall
Now 4 ft wide × 16 ft tall
🟥 Screen direction flips:
Was running LEFT → RIGHT
Now running TOP → BOTTOM
🧑 People face left/right instead of up/down
Rule: The screen always runs along the long axis of the cluster — parallel to the join line between the two rows of desks.
The screen direction is determined by cluster orientation, not a fixed direction. Horizontal cluster → screens run L→R. Vertical cluster → screens run T→B. The screen panels always sit on top of the centre join line, whatever direction that line faces.
